Talk to the demo agent. Each reply shows the predicted intent and the model's confidence, so you can see why the agent picked its response — not just what it said.

Demo intents: Greeting, Good, Bad, Bye, IAmFrom. Try “hi”, “I am fine”, “I am from Bulgaria”, “bye”.

Define your own intents with a few example phrases each, then type a sentence and see which intent the model picks. Everything is trained in‑memory per request — no persistence.

Aggregate behavioural signals across every chat session served by this demo — no conversation IDs, no message text. Useful to see where the agent hesitates and which intents need more examples.
